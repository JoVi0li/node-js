
# NodeJS

Este repositório contém diversos projetos desenvolvidos utilizando NodeJS, bem como, bibliotecas e frameworks que o compõe.

O desenvolvimento de tais projetos tem o objetivo de aprendizagem.

Por isso, sinta-se mais do que à vontade para sugerir quaisquer melhorias, mudanças e correções :heart:.


## Projetos

- [Fundamentos](https://github.com/JoVi0li/node-js/tree/main/fundamentos)
    - Introdução ao NodeJS.

- [FinAPI](https://github.com/JoVi0li/node-js/tree/main/finapi)
    - API que disponibiliza um CRUD para que o usuário gerencie sua vida financeira.

- [RentalX](https://github.com/JoVi0li/node-js/tree/main/rentalx)
    - Uma API para o aluguel de carros.
